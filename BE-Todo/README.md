# TS CRUD Project

A TypeScript project implementing CRUD operations for a Task entity using MongoDB.

## Prerequisites
- MongoDB running locally on default port (27017)

## Entity Fields
- title: string
- comment: string
- tag: string
- date: string
- completed: boolean

## API Endpoints
- GET /tasks - Retrieve all tasks
- POST /tasks - Create a new task
- PUT /tasks/:id - Update a task by ID
- DELETE /tasks/:id - Delete a task by ID

## Installation
npm install

## Development
npm run dev

## Build
npm run build

## Start
npm start
